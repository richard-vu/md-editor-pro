## Usage

### Export Commands

Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS) to open the Command Palette, then:

- **MD Editor Pro: Export PDF (Current File)** - Export current file to PDF
- **MD Editor Pro: Export PDF (Folder)** - Export entire folder to PDF
- **MD Editor Pro: Export HTML (Current File)** - Export current file to HTML
- **MD Editor Pro: Export HTML (Folder)** - Export entire folder to HTML

### Translation Commands

Select text in a Markdown file, then use:
- **VN to EN** - Translate Vietnamese to English
- **EN to VN** - Translate English to Vietnamese
- **VN to JP** - Translate Vietnamese to Japanese
- **JP to VN** - Translate Japanese to Vietnamese
- **EN to JP** - Translate English to Japanese
- **JP to EN** - Translate Japanese to English
